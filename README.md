# Mini Novel Project (Facade Pattern)

## 📌 Описание
This project is a simple visual novel developed in Python.  
It demonstrates the use of the **Facade design pattern** to simplify the interaction between different game systems (audio, scenes, dialogue, saving system).

## 🎮 Features
- Scene switching system
- Dialogue system
- Audio management
- Save system
- Simple visual novel structure

## 🧠 Design Pattern Used
**Facade Pattern**  
Used to provide a single interface (`GameFacade`) that controls all subsystems of the game.

## 🛠 Technologies
- Python
- Pygame (if used)
- Object-Oriented Programming

## 🚀 How to run
1. Install Python 3
2. Install dependencies (if needed):

3. pip install pygame

3. Run the project:

python main.py
